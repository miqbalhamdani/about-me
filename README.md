# Iqbal Hamdani Portfolio

Personal portfolio for Iqbal Hamdani, built with Next.js App Router, TypeScript, and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site locally.

## SEO Configuration

Set `NEXT_PUBLIC_SITE_URL` to the production origin before building or deploying. This value powers canonical URLs, Open Graph metadata, `robots.txt`, and `sitemap.xml`.

```bash
NEXT_PUBLIC_SITE_URL=https://your-domain.com npm run build
```

## Checks

```bash
npm run lint
NEXT_PUBLIC_SITE_URL=https://your-domain.com npm run build
```
